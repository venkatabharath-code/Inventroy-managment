# Hospital Management System

A modern Hospital Management System built using React and Vite, designed to streamline hospital operations through inventory tracking, procurement management, and asset management. The application provides an intuitive dashboard experience for monitoring resources, managing stock levels, tracking purchase orders, and maintaining hospital assets efficiently.

## Installation

Clone the repository:

```bash
git clone https://github.com/venkatabharath-code/hospital-managment.git
```

Navigate to the project:

```bash
cd hospital-managment
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run lint checks:

```bash
npm run lint
```
