# Hospital Inventory Management System

A modern, responsive, and robust web application designed to streamline hospital inventory, procurement, and asset management. Built with React and Vite, this dashboard provides real-time statistics, low-stock alerts, category distributions, and recent stock movements through an intuitive, aesthetic user interface.

🚀 **Live Demo:** [View the Live Application Here](https://venkatabharath-code.github.io/Inventroy-managment/)

## Features
* **Inventory Dashboard:** Real-time metrics for total SKUs, low stock warnings, and soon-to-expire items.
* **Stock Management:** Seamlessly add, edit, and track medical supplies, equipment, and consumables.
* **Procurement:** Generate and track purchase orders with suppliers easily.
* **Asset Tracking:** Monitor expensive hospital machinery, maintenance schedules, and statuses.
* **Data Persistence:** Local storage integration ensures your data is saved securely between sessions.
